# ✦ Noctura

A responsive front-end web layout built with **HTML**, **Bootstrap 5**, and **custom CSS** — recreated from a Figma design reference. Features a dark, editorial aesthetic with a space/earth hero section, portfolio-style image showcases, and a blog card grid.

---

## ✨ Features

- **Full-screen hero section** with a masked background image and bold typography
- **Responsive navbar** with mobile hamburger toggle (Bootstrap)
- **Logo/brand icon row** using Font Awesome social icons
- **Large image showcase** sections for featured work categories
- **Blog card grid** — 4-column responsive layout with consistent card styling
- **Dark green color palette** (`#061a0e`, `#164c2b`) for a moody, editorial feel
- Smooth gradient mask on the hero using CSS `mask-image`

## 🚀 Getting Started

No build tools required — just open it in a browser.
```bash
git clone https://github.com/ClydeCenteno1/Noctura.git
cd Noctura
open index.html
```

Or drag `index.html` straight into any browser window.

## 🎨 Design Reference

This project is a **Figma-to-code** recreation of the [Simple Website Design](https://www.figma.com/proto/UfPJ6yIIfrGjV1EdiimHmw/Simple-website-design--Community-?node-id=2-2&p=f&t=8Z0kQjjGi0W9e3UZ-0&scaling=min-zoom&content-scaling=fixed&page-id=0%3A1) community template. The goal was to practice translating a visual design into clean, responsive HTML and CSS as faithfully as possible.
